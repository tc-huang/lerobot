ACT 是一種**輕量且高效的模仿學習策略**，特別適合精細操作任務。由於其訓練速度快、計算需求低且表現優異，它是**我們推薦您入門 LeRobot 時首選的模型**。

## 模型概覽

Action Chunking with Transformers（ACT）由 Zhao 等人在論文 [Learning Fine-Grained Bimanual Manipulation with Low-Cost Hardware](https://arxiv.org/abs/2304.13705) 中提出。該策略旨在使用平價硬體和最少的示範資料來實現精確且充分接觸（contact-rich）的操作任務。

### 為什麼 ACT 非常適合初學者

ACT 作為絕佳的入門選擇，有以下幾個原因：

- **訓練速度快**：在單張 GPU 上幾小時內即可完成訓練
- **輕量化**：僅約 8,000 萬個參數，高效且易於使用
- **資料效率高**：通常僅需 50 筆示範即可達到高成功率

### 架構

ACT 使用基於 Transformer 的架構，包含三個主要組件：

1. **視覺骨幹網路**：ResNet-18 處理來自多個攝影機視角的影像
2. **Transformer 編碼器**：整合來自攝影機特徵、關節位置和學習到的潛在變數的資訊
3. **Transformer 解碼器**：使用交叉注意力機制（cross-attention）生成連貫的動作序列

策略的輸入包括：

- 多張 RGB 影像（例如來自手腕攝影機、前方/頂部攝影機）
- 當前機器人關節位置
- 潛在風格變數 `z`（訓練期間學習，推論時設為零）

輸出為一個包含 `k` 個未來動作序列的區塊。

## 訓練 ACT

ACT 可與標準的 LeRobot 訓練流程無縫整合。以下是在您的資料集上訓練 ACT 的完整範例：

```bash
lerobot-train \
  --dataset.repo_id=${HF_USER}/your_dataset \
  --policy.type=act \
  --output_dir=outputs/train/act_your_dataset \
  --job_name=act_your_dataset \
  --policy.device=cuda \
  --wandb.enable=true \
  --policy.repo_id=${HF_USER}/act_policy
```

### 訓練技巧

1. **從預設值開始**：ACT 的預設超參數對大多數任務都能良好運作
2. **訓練時長**：在單張 GPU 上進行 100k 訓練步驟預計需要幾小時
3. **批次大小**：建議從批次大小 8 開始，再根據您的 GPU 記憶體進行調整

### 使用 Google Colab 訓練

如果您的本機電腦沒有強大的 GPU，可以按照 [ACT 訓練筆記本](./notebooks#training-act) 使用 Google Colab 來訓練您的模型。

## 評估 ACT

訓練完成後，您可以使用 `lerobot-record` 指令搭配已訓練的策略來評估您的 ACT 策略。這將執行推論並記錄評估回合：

```bash
lerobot-record \
  --robot.type=so100_follower \
  --robot.port=/dev/ttyACM0 \
  --robot.id=my_robot \
  --robot.cameras="{ front: {type: opencv, index_or_path: 0, width: 640, height: 480, fps: 30}}" \
  --display_data=true \
  --dataset.repo_id=${HF_USER}/eval_act_your_dataset \
  --dataset.num_episodes=10 \
  --dataset.single_task="Your task description" \
  --dataset.streaming_encoding=true \
  --dataset.encoder_threads=2 \
  # --dataset.vcodec=auto \
  --policy.path=${HF_USER}/act_policy
```
