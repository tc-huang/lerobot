# GR00T N1.5 策略

GR00T N1.5 是 NVIDIA 設計的開放基礎模型（foundation model），專為通用人形機器人推理與技能而打造。它是一個跨具身模型（cross-embodiment model），接受多模態輸入（包括語言和影像），以在多樣化的環境中執行操作任務。

本文件概述了其在 LeRobot 框架中的整合與使用細節。

## 模型概述

NVIDIA Isaac GR00T N1.5 是 GR00T N1 基礎模型的升級版本。它旨在提升人形機器人的泛化能力與語言理解遵循能力。

開發者和研究人員可以使用自己的真實或合成資料對 GR00T N1.5 進行後訓練，以適應特定的人形機器人或任務。

GR00T N1.5（具體來說是 GR00T-N1.5-3B 模型）使用預訓練的視覺和語言編碼器建構而成。它利用 flow matching action transformer 來對動作塊（chunk of actions）建模，並以視覺、語言和本體感覺作為條件。

其優異的效能來自於在大規模且多樣化的人形機器人資料集上進行訓練，這些資料集包括：

- 從機器人擷取的真實資料。
- 使用 NVIDIA Isaac GR00T Blueprint 生成的合成資料。
- 網際網路規模的影片資料。

這種方法使模型透過後訓練能夠高度適應特定的具身類型、任務和環境。

## 安裝需求

截至目前，GR00T N1.5 的內部運作需要 Flash Attention。

我們正在努力將此設為可選項，但目前這意味著需要額外的安裝步驟，且只能在啟用 CUDA 的裝置上使用。

1. 按照我們[安裝指南](./installation)中的環境設定進行操作。**注意**：在此步驟中不要安裝 `lerobot`。
2. 執行以下命令安裝 [Flash Attention](https://github.com/Dao-AILab/flash-attention)：

```bash
# 請查看 https://pytorch.org/get-started/locally/ 以取得適合您系統的版本
pip install "torch>=2.2.1,<2.8.0" "torchvision>=0.21.0,<0.23.0" # --index-url https://download.pytorch.org/whl/cu1XX
pip install ninja "packaging>=24.2,<26.0" # flash attention 依賴項
pip install "flash-attn>=2.5.9,<3.0.0" --no-build-isolation
python -c "import flash_attn; print(f'Flash Attention {flash_attn.__version__} imported successfully')"
```

3. 執行以下命令安裝 LeRobot：

```bash
pip install lerobot[groot]
```

## 使用方式

若要在 LeRobot 設定中使用 GR00T，請將策略類型指定為：

```python
policy.type=groot
```

## 訓練

### 訓練指令範例

以下是在您自己的資料集上微調基礎 GR00T 模型的完整訓練指令：

```bash
# 使用多 GPU 設定
accelerate launch \
  --multi_gpu \
  --num_processes=$NUM_GPUS \
  $(which lerobot-train) \
  --output_dir=$OUTPUT_DIR \
  --save_checkpoint=true \
  --batch_size=$BATCH_SIZE \
  --steps=$NUM_STEPS \
  --save_freq=$SAVE_FREQ \
  --log_freq=$LOG_FREQ \
  --policy.push_to_hub=true \
  --policy.type=groot \
  --policy.repo_id=$REPO_ID \
  --policy.tune_diffusion_model=false \
  --dataset.repo_id=$DATASET_ID \
  --wandb.enable=true \
  --wandb.disable_artifact=true \
  --job_name=$JOB_NAME
```

## 效能結果

### Libero 基準測試結果

> [!NOTE]
> 請參考我們的 Libero 使用說明：[Libero](./libero)

GR00T 在 Libero 基準測試套件上展現了優異的效能。為了比較和測試其 LeRobot 實作，我們在 Libero 資料集上對 GR00T N1.5 模型進行了 30k 步的微調，並將結果與 GR00T 參考結果進行比較。

| 基準測試           | LeRobot 實作 | GR00T 參考值 |
| ------------------ | ------------ | ------------ |
| **Libero Spatial** | 82.0%        | 92.0%        |
| **Libero Object**  | 99.0%        | 92.0%        |
| **Libero Long**    | 82.0%        | 76.0%        |
| **平均**           | 87.0%        | 87.0%        |

這些結果展示了 GR00T 在各種機器人操作任務中強大的泛化能力。若要重現這些結果，您可以參考 [Libero](https://huggingface.co/docs/lerobot/libero) 章節中的說明。

### 在您的硬體環境中進行評估

使用您的參數訓練模型完成後，您可以在下游任務中執行推論。請參考[機器人的模仿學習](./il_robots)中的說明。例如：

```bash
lerobot-record \
  --robot.type=bi_so_follower \
  --robot.left_arm_port=/dev/ttyACM1 \
  --robot.right_arm_port=/dev/ttyACM0 \
  --robot.id=bimanual_follower \
  --robot.cameras='{ right: {"type": "opencv", "index_or_path": 0, "width": 640, "height": 480, "fps": 30},
    left: {"type": "opencv", "index_or_path": 2, "width": 640, "height": 480, "fps": 30},
    top: {"type": "opencv", "index_or_path": 4, "width": 640, "height": 480, "fps": 30},
  }' \
  --display_data=true \
  --dataset.repo_id=<user>/eval_groot-bimanual  \
  --dataset.num_episodes=10 \
  --dataset.single_task="Grab and handover the red cube to the other arm" \
  --dataset.streaming_encoding=true \
  --dataset.encoder_threads=2 \
  # --dataset.vcodec=auto \
  --policy.path=<user>/groot-bimanual \ # 您訓練好的模型
  --dataset.episode_time_s=30 \
  --dataset.reset_time_s=10
```
